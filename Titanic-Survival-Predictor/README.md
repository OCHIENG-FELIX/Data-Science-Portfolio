# Titanic Survival Predictor

**End-to-End Machine Learning Web Application**

This project predicts whether a passenger would survive the Titanic disaster based on features like age, gender, class, fare, etc.

### Features
- Data cleaning and feature engineering
- Multiple machine learning models trained and evaluated
- Interactive web application built with Streamlit
- Model deployment on Streamlit Cloud

### Technologies Used
- Python
- Pandas, Scikit-learn
- Streamlit (for web interface)
